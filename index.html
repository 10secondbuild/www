How it Works?

REST / WebHooks

All builds are triggered via WebHooks into the REST API. No polling

Input

POSTed message are piped into container via STDIN and CGI interface.

GitHub / BitBucket

We provide example Docker images for simple integration by processing the GitHib push event json or fork it

Output

The Docker containers volume is used to declare the output of the build

Pipelines

Additional stages can be created by chaining together output volumes with other containers

S3 support / GitHub Release

We provide pre built end stages for releasing your binaries or fork them
